Dashboard adalah control panel web untuk mengelola konfigurasi utama LumigiaBOT secara visual. Beberapa halaman sudah mendukung perubahan langsung dari web, sementara beberapa halaman lain berfungsi sebagai monitoring agar admin bisa membaca data server tanpa membuka database.

## Cara Akses

<Steps>
<Step>
### Buka Dashboard

Kunjungi halaman dashboard LumigiaBOT di browser kamu.
</Step>

<Step>
### Login dengan Discord

Klik tombol **Login** dan autentikasi menggunakan akun Discord kamu. Dashboard menggunakan Discord OAuth2 untuk memverifikasi identitas dan permission kamu.
</Step>

<Step>
### Pilih Server

Setelah login, kamu akan melihat daftar server yang bisa kamu kelola. Pilih server yang ingin dikonfigurasi.

Hanya server di mana kamu memiliki permission **Manage Server** yang akan muncul.
</Step>
</Steps>

## Fitur Dashboard

### Settings

Konfigurasi umum server, termasuk:
- **Bahasa** - Pilih bahasa bot (Indonesia / English)
- **Welcome message** - Atur pesan sambutan untuk member baru
- **Channel log** - Pilih channel untuk log moderasi dan AutoMod
- **Konfigurasi tiket** - Atur category, role support, channel log, batas tiket, dan auto-close
- **Anti-raid** - Aktifkan proteksi raid dan sesuaikan threshold
- **Warning escalation** - Pilih preset tindakan otomatis setelah warning tertentu

### AutoMod

Kelola filter auto-moderasi server:
- Toggle filter **on/off** dengan satu klik
- Pilih **aksi** yang diambil untuk setiap filter (delete, warn, mute, kick, ban)

### Voice

Konfigurasi fitur voice channel otomatis:
- **Hub channel** - Channel yang di-join untuk membuat voice sementara
- **Category** - Category tempat voice channel dibuat
- **Nama default** - Template nama untuk voice channel baru
- **Limit** - Batas member per voice channel

### Leveling

Atur sistem leveling server:
- **XP rate** - Jumlah XP per pesan
- **Cooldown** - Jeda antar pemberian XP
- **Multiplier** - Pengali XP untuk event khusus
- **Channel pengumuman** - Channel untuk notifikasi level up

### Streams

Kelola notifikasi live stream:
- Tambah notifikasi untuk streamer Twitch atau YouTube
- Hapus notifikasi yang tidak diperlukan
- Atur channel notifikasi, role ping, dan pesan kustom

### Schedule

Kelola jadwal streaming mingguan:
- Tambah jadwal baru dengan hari, waktu, timezone, judul, dan deskripsi
- Hapus jadwal yang sudah tidak relevan

### Fan Art

Konfigurasi sistem fan art gallery:
- Atur channel submit dan channel gallery
- Toggle approval system on/off
- Lihat submission terbaru dan hapus karya yang sudah di-approve jika diperlukan

### Tickets

Pantau sistem tiket:
- Lihat total tiket, tiket open, claimed, dan closed
- Buka transcript HTML untuk tiket yang sudah ditutup
- Konfigurasi tiket tetap berada di **Settings**, sedangkan pembuatan panel tiket dilakukan melalui command `/ticket-setup`

### Reaction Roles

Pantau panel reaction role yang sudah dibuat:
- Lihat daftar panel, mode, jumlah entry, channel, dan status posting
- Buat, edit, dan hapus panel melalui command `/reaction-role`

### Giveaways

Pantau giveaway server:
- Lihat total giveaway, status active/ended, hadiah, jumlah pemenang, host, dan waktu selesai
- Mulai, akhiri, atau reroll giveaway melalui command `/giveaway`

### Moderation & Logs

Audit data moderasi:
- Lihat warning yang pernah diberikan di server
- Lihat audit log tindakan moderasi dan bot

### Analytics

Lihat statistik server secara visual (read-only):
- Ringkasan pesan, join, leave, dan active users dalam 7 hari terakhir
- Breakdown harian
- Top channel berdasarkan jumlah pesan

## Keunggulan Dashboard vs Commands

| Dashboard | Commands |
|---|---|
| Dropdown channel & role otomatis — tidak perlu copy-paste ID | Harus tahu ID channel atau mention manual |
| Auto-save perubahan setiap kali toggle atau pilih opsi | Harus ketik ulang command jika ada kesalahan |
| Visual yang mudah dipahami dengan preview langsung | Output berupa teks di chat |
| Konfigurasi inti dan monitoring server di satu tempat | Harus ingat berbagai command berbeda |

<Callout type="info">
Kamu harus memiliki permission **Manage Server** di Discord untuk bisa mengakses dan mengonfigurasi server melalui dashboard. Jika server tidak muncul di daftar, pastikan kamu memiliki permission yang cukup.
</Callout>
