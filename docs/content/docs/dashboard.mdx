---

Dashboard adalah antarmuka web untuk mengonfigurasi semua fitur LumigiaBOT secara visual. Tidak perlu menghafal command — cukup klik, toggle, dan simpan.

## Cara Akses

<Steps>
<Step>
### Buka Dashboard

Kunjungi halaman dashboard LumigiaBOT di browser kamu.
</Step>

<Step>
### Login dengan Discord

Klik tombol **Login** dan autentikasi menggunakan akun Discord kamu. Dashboard menggunakan Discord OAuth2 untuk memverifikasi identitas dan permission kamu.
</Step>

<Step>
### Pilih Server

Setelah login, kamu akan melihat daftar server yang bisa kamu kelola. Pilih server yang ingin dikonfigurasi.

Hanya server di mana kamu memiliki permission **Manage Server** yang akan muncul.
</Step>
</Steps>

## Fitur Dashboard

### Settings

Konfigurasi umum server, termasuk:
- **Bahasa** — Pilih bahasa bot (Indonesia / English)
- **Welcome message** — Atur pesan sambutan untuk member baru
- **Channel log** — Pilih channel untuk log aktivitas
- **Konfigurasi tiket** — Atur category dan channel untuk sistem ticketing

### AutoMod

Kelola filter auto-moderasi server:
- Toggle filter **on/off** dengan satu klik
- Pilih **aksi** yang diambil untuk setiap filter (hapus pesan, warn, mute, dll.)

### Voice

Konfigurasi fitur voice channel otomatis:
- **Hub channel** — Channel yang di-join untuk membuat voice sementara
- **Category** — Category tempat voice channel dibuat
- **Nama default** — Template nama untuk voice channel baru
- **Limit** — Batas member per voice channel

### Leveling

Atur sistem leveling server:
- **XP rate** — Jumlah XP per pesan
- **Cooldown** — Jeda antar pemberian XP
- **Multiplier** — Pengali XP untuk event khusus
- **Channel pengumuman** — Channel untuk notifikasi level up

### Streams

Kelola notifikasi live stream:
- **Tambah** notifikasi untuk streamer Twitch atau YouTube
- **Hapus** notifikasi yang tidak diperlukan
- Konfigurasi pesan dan channel notifikasi

### Schedule

Kelola jadwal streaming mingguan:
- **Tambah** jadwal baru dengan hari, waktu, dan judul
- **Hapus** jadwal yang sudah tidak relevan
- Preview embed jadwal sebelum dikirim

### Fan Art

Konfigurasi sistem fan art gallery:
- Atur **channel submit** dan **channel gallery**
- Toggle **approval system** on/off
- Konfigurasi pengaturan gallery lainnya

### Analytics

Lihat statistik server secara visual (read-only):
- Grafik pesan harian
- Statistik member join/leave
- Aktivitas per channel

## Keunggulan Dashboard vs Commands

| Dashboard | Commands |
|---|---|
| Dropdown channel & role otomatis — tidak perlu copy-paste ID | Harus tahu ID channel atau mention manual |
| Auto-save perubahan setiap kali toggle atau pilih opsi | Harus ketik ulang command jika ada kesalahan |
| Visual yang mudah dipahami dengan preview langsung | Output berupa teks di chat |
| Semua konfigurasi di satu tempat | Harus ingat berbagai command berbeda |

<Callout type="info">
Kamu harus memiliki permission **Manage Server** di Discord untuk bisa mengakses dan mengonfigurasi server melalui dashboard. Jika server tidak muncul di daftar, pastikan kamu memiliki permission yang cukup.
</Callout>
