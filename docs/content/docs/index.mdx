# Selamat Datang di LumigiaBOT

**LumigiaBOT** adalah bot Discord all-in-one yang dirancang untuk membantu kamu mengelola server dengan mudah dan efisien. Mulai dari moderasi, auto-mod, ticketing, anti-raid, leveling, giveaway, hingga fitur khusus streamer — semua tersedia dalam satu bot.

Bot ini mendukung konfigurasi independen per server, sehingga setiap server bisa disesuaikan sesuai kebutuhannya masing-masing.

## Fitur Utama

<Cards>
  <Card title="🛡️ Moderasi" href="/docs/moderation">
    Ban, kick, mute, warn, purge, slowmode, dan lockdown. Dilengkapi sistem eskalasi peringatan otomatis dan perlindungan anti-raid.
  </Card>
  <Card title="🤖 AutoMod" href="/docs/automod">
    Filter otomatis untuk spam, link, kata terlarang, caps berlebihan, emoji spam, dan mention massal. Semuanya bisa dikustomisasi.
  </Card>
  <Card title="🎫 Tickets" href="/docs/tickets">
    Sistem tiket support lengkap dengan transkrip otomatis, auto-close, dan statistik tiket per server.
  </Card>
  <Card title="🔊 Voice" href="/docs/voice">
    Fitur Join-to-Create untuk channel suara sementara. Atur nama, limit, lock/unlock, dan permission langsung dari perintah.
  </Card>
  <Card title="⭐ Leveling" href="/docs/leveling">
    Sistem XP dan level dengan leaderboard, role rewards otomatis, dan progress bar visual.
  </Card>
  <Card title="🎬 Streamer" href="/docs/streamer">
    Notifikasi stream Twitch/YouTube, galeri fan art dengan sistem approval, dan analytics server.
  </Card>
</Cards>
