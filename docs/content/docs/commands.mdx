# Daftar Perintah

Berikut adalah semua slash commands yang tersedia di LumigiaBOT, dikelompokkan berdasarkan kategori.

<Callout type="info">
  Semua perintah menggunakan Slash Commands (`/`). Ketik `/` di chat Discord untuk melihat daftar perintah beserta opsinya.
</Callout>

## Admin

| Perintah | Deskripsi | Permission |
|---|---|---|
| `/setup` | Konfigurasi awal bot di server | Manage Server |
| `/settings` | Mengubah pengaturan server (bahasa, log channel, dll.) | Manage Server |
| `/audit-log` | Melihat riwayat log audit server | Manage Server |

## Moderasi

| Perintah | Deskripsi | Permission |
|---|---|---|
| `/ban` | Banned member dari server | Ban Members |
| `/kick` | Kick member dari server | Kick Members |
| `/mute` | Timeout/mute member | Moderate Members |
| `/warn` | Memberikan peringatan ke member | Moderate Members |
| `/purge` | Menghapus pesan secara massal | Manage Messages |
| `/slowmode` | Mengatur slowmode di channel | Manage Channels |
| `/lockdown` | Mengunci channel dari aktivitas member | Manage Channels |

## Voice

Semua subcommands menggunakan prefix `/voice`.

| Perintah | Deskripsi | Permission |
|---|---|---|
| `/voice setup` | Mengatur channel Join-to-Create | Manage Channels |
| `/voice settings` | Mengubah pengaturan voice default | Manage Channels |
| `/voice name` | Mengubah nama channel voice sementara | Pemilik Channel |
| `/voice limit` | Mengatur batas user di channel voice | Pemilik Channel |
| `/voice lock` | Mengunci channel voice | Pemilik Channel |
| `/voice unlock` | Membuka kunci channel voice | Pemilik Channel |
| `/voice permit` | Mengizinkan user masuk channel terkunci | Pemilik Channel |
| `/voice reject` | Menolak/mengeluarkan user dari channel | Pemilik Channel |
