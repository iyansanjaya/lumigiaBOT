# Daftar Perintah

Berikut adalah semua slash commands yang tersedia di LumigiaBOT, dikelompokkan berdasarkan kategori.

<Callout type="info">
  Semua perintah menggunakan Slash Commands (`/`). Ketik `/` di chat Discord untuk melihat daftar perintah beserta opsinya.
</Callout>

## Admin

| Perintah | Deskripsi | Permission |
|---|---|---|
| `/setup` | Konfigurasi awal bot di server | Manage Server |
| `/settings view` | Melihat pengaturan server saat ini | Manage Server |
| `/settings language` | Mengubah bahasa bot per server | Manage Server |
| `/settings mod-log` | Mengatur channel log moderasi | Manage Server |
| `/audit-log view` | Melihat 10 audit log terbaru | Manage Server |
| `/audit-log search` | Mencari audit log berdasarkan user | Manage Server |

## Moderasi

| Perintah | Deskripsi | Permission |
|---|---|---|
| `/ban` | Banned member dari server | Ban Members |
| `/kick` | Kick member dari server | Kick Members |
| `/mute` | Timeout/mute member | Moderate Members |
| `/warn` | Memberikan peringatan ke member | Moderate Members |
| `/warnings` | Melihat riwayat warning member | Moderate Members |
| `/clearwarns` | Menghapus semua atau satu warning tertentu | Moderate Members |
| `/purge` | Menghapus pesan massal, opsional per user | Manage Messages |
| `/slowmode` | Mengatur slowmode di channel | Manage Channels |
| `/lockdown` | Mengunci atau membuka channel | Manage Channels |
